Object and Embed Elements - Further reading

This is a test content to check working of the object element and understand its difference with respect to the iframe element.
The iframe element embeds an html or another webpage into the parent webpage where it is used. The width and height attributes are used
in the iframe tags to define the dimension of the embedded web-page. Hence, when the content of the embedded-webpage exceeds the height 
and/or width of the iframe, vertical and/or horizontal scrolling bars automatically appearas applicable. The object element works the 
same way as iframe w.r.t the automatic appearance of the scrollbars according to the dimension of object element and web-page content.
